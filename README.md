# Proyecto COVID Colombia

Aplicación en Python que consulta datos de COVID-19 en Colombia mediante una API
y los muestra en una interfaz gráfica.

## Tecnologías usadas
- Python
- Requests
- Tkinter

## Cómo ejecutar
1. Instalar dependencias:
   pip install -r requirements.txt

2. Ejecutar:
   python principal.py
