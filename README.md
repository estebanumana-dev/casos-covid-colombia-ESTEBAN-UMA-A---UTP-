# Proyecto COVID Colombia

Aplicación en Python que consulta datos de COVID-19 en Colombia mediante una API
y los muestra en una interfaz gráfica.

## Tecnologías usadas
- Python
- Requests
- Tkinter

## Requisitos
- Python 3.10 o superior
- pip

# Instalación

Clonar el repositorio:

git clone https://github.com/estebanumana-dev/casos-covid-colombia-ESTEBAN-UMA-A---UTP-.git

Entrar a la carpeta:

cd casos-covid-colombia-ESTEBAN-UMA-A---UTP-

Instalar dependencias:

pip install -r requirements.txt

## Ejecución

python main.py

## Uso

1. Ingresar el nombre del departamento
2. El sistema valida los datos
3. Muestra los resultados o un mensaje de error si el dato no es válido

2. Ejecutar:
   python principal.py
